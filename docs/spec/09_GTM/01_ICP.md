# Ideal Customer Profile

## Best first customers
- 500-5,000 employees.
- Microsoft 365 E5 or Microsoft-heavy environment.
- Copilot pilot or rollout.
- Sensitive SharePoint/OneDrive/Teams data.
- Security/compliance approval.
- M365 admin and security team.

## Best industries
- Financial services.
- Insurance.
- Healthcare.
- Legal.
- Professional services.
- B2B SaaS.
- Government contractors.
- Research/education with sensitive data.

## Buyer titles
- CISO.
- Head of Information Security.
- VP Security Engineering.
- Director of Security Architecture.
- CIO.
- Head of IT.
- Director of Microsoft 365.
- Data Governance Lead.

## Trigger events
- Copilot rollout.
- Copilot Studio project.
- Purview/SAM implementation.
- Security audit.
- Board AI risk question.
- Hiring for M365 governance/security.
