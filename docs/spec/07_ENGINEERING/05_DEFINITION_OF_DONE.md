# Definition of Done

## Feature done
- Works in demo.
- Has tests.
- Handles errors.
- Logs audit event if sensitive.
- Has empty/loading/error states.
- Is documented.
- Does not increase Graph scopes without review.

## Finding done
- Severity.
- Score.
- Evidence.
- Permission/agent path.
- Remediation.
- Exported in report.

## Report done
- Executive summary.
- Scope/methodology.
- Critical findings.
- Remediation.
- Limitations.
- Export works.

## Pilot-ready
- Security packet complete.
- Graph scopes documented.
- Deletion tested.
- Audit logs working.
- Customer-cloud path drafted.
