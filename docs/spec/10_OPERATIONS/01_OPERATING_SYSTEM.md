# Operating System

## Weekly cadence
Monday: metrics, goals, risks.
Tuesday-Thursday: build, calls, demos, outbound.
Friday: demo progress, review customer feedback, review spend, decide.

## Daily checklist
- Did we talk to a buyer/user?
- Did we improve demo?
- Did we learn something that could kill the idea?
- Did we keep scope tight?
- Did we avoid unnecessary spend?

## Weekly metrics
- Outbound sent.
- Replies.
- Calls booked/completed.
- Demo requests.
- Price discussions.
- Pilot next steps.
- Product milestones.
- Azure spend.
- Top risk.
