# MVP User Stories

## Epic A - Workspace and demo data
- As a founder, I can seed a fake Microsoft 365 company.
- As a user, I can create a workspace.
- As a user, I can run demo mode without Microsoft access.

## Epic B - Metadata connector
- As an admin, I can review required Microsoft permissions.
- As an admin, I can connect a sandbox tenant.
- As a worker, I can ingest users, groups, sites, drives, files, and permission metadata.

## Epic C - Rule engine
- As a security user, I can detect sensitive file names/paths/labels.
- As a security user, I can detect broad access.
- As a security user, I can detect risky links, guests, and broad groups.
- As a security user, I can detect simulated risky agent actions.

## Epic D - Scenario engine
- As a security user, I can run employee, contractor, broad access, guest, and agent scenarios.
- As a security user, I can see the exposure path produced by each scenario.

## Epic E - Dashboard and report
- As a security user, I can view findings by severity.
- As a security user, I can inspect evidence.
- As a buyer, I can export a report.
