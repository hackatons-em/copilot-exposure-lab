# Core Workflows

## First exposure assessment
1. Create workspace.
2. Choose demo, sandbox, or customer tenant.
3. Review permissions.
4. Connect or seed data.
5. Run metadata scan.
6. Build permission graph.
7. Run scenarios.
8. Generate findings.
9. Export report.

## Finding review
1. Open finding.
2. See summary, severity, score.
3. Inspect evidence chain.
4. Review affected users/resources.
5. Read remediation.
6. Assign owner.
7. Mark status.
8. Rerun after fix.

## Proof-of-fix
1. Admin fixes link, permission, label, owner, or agent setting in Microsoft tools.
2. Product reruns scenario.
3. Old exposure path is compared to new result.
4. Report shows fixed/residual risk.

## Design partner assessment
1. Agree scope.
2. Run metadata-only scan.
3. Review findings.
4. Choose top fixes.
5. Export report.
6. Optionally rerun after remediation.
