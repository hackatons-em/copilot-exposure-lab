# What We Are Building

## Name placeholder
Copilot Exposure Lab

## One-line pitch
A Microsoft-native exposure testing tool that safely simulates how Copilot, Copilot Studio agents, and Microsoft 365 permissions could expose sensitive data, then tells security teams exactly what to fix.

## MVP product promise
In one afternoon, show the top ways a company's Copilot rollout could expose sensitive Microsoft 365 data and what to fix first.

## Primary modules
1. Tenant/demo connector.
2. Microsoft Graph metadata connector.
3. Permission graph.
4. Exposure scenario engine.
5. Risk scoring engine.
6. Evidence store.
7. Findings dashboard.
8. Remediation tracker.
9. Report generator.
10. Audit log.

## MVP user workflow
1. Create workspace.
2. Seed demo data or connect sandbox tenant.
3. Run exposure scenario.
4. Review findings.
5. Open evidence chain.
6. Export report.
7. Rerun after fix.

## V1 user workflow
1. Customer approves narrow scope.
2. Scanner ingests metadata only.
3. Product runs scenarios.
4. Security reviews evidence.
5. Admin remediates in Microsoft tools.
6. Product reruns and proves fix.
