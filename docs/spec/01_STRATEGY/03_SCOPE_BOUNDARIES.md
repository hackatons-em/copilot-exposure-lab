# Scope Boundaries

## Build in MVP
- Demo dataset.
- Local and cloud demo.
- Microsoft Graph metadata spike.
- SharePoint/OneDrive metadata ingestion.
- Users, groups, sites, drives, files, permissions.
- Deterministic exposure rules.
- Dashboard.
- Report export.
- Audit logs.
- Cost controls.

## Do not build in MVP
- Full Purview replacement.
- Full Defender replacement.
- Full Copilot Studio runtime protection.
- Continuous production monitoring.
- Automatic remediation.
- Multi-cloud.
- SOC 2.
- Customer production-scale indexing.
- Full document-content ingestion by default.

## V1 after validation
- Customer tenant onboarding.
- Metadata-only assessment.
- Customer-cloud deployment draft.
- Re-scan/proof-of-fix.
- Pilot security packet.

## V2 after paid pilots
- Copilot Studio agent inventory.
- Power Platform connector risk checks.
- Defender/Purview/Sentinel exports.
- Jira/ServiceNow task sync.
- Scheduled scans.

## V3 after traction
- Continuous exposure management.
- Multi-system AI exposure graph.
- Custom RAG permission testing.
- Agent action audit and controls.
