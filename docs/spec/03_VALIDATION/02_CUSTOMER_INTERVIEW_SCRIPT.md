# Customer Interview Script

## Opening
"We are researching how Microsoft 365 Copilot and Copilot Studio rollouts are reviewed from a data exposure and security perspective. I want to understand what you already do and where it breaks."

## Questions
1. Are you using or planning Microsoft 365 Copilot?
2. Are you using Copilot Studio or Power Platform agents?
3. Who owns security approval?
4. Which data sources are in scope?
5. What was the last security concern raised?
6. How do you check SharePoint/OneDrive oversharing today?
7. Which Microsoft tools do you use?
8. What do those tools answer well?
9. What do they not answer well?
10. Have rollouts been delayed?
11. Who fixes risky permissions?
12. How long does remediation take?
13. How do you prove a fix worked?
14. What report does the CISO need?
15. Would metadata-only assessment be useful?
16. Would customer-cloud be required?
17. What would make you reject a startup here?
18. What budget would pay for this?
19. Would $10k-$25k be unreasonable, reasonable, or cheap?
20. Who else should we talk to?

## After demo
- Which finding felt real?
- Which finding felt fake?
- What evidence is missing?
- Would this report help?
- Would you run this on a scoped environment?
