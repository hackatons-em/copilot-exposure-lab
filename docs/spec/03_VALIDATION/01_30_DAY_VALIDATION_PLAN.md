# 30-Day Validation Plan

## Goal
Decide whether this deserves serious build investment.

## Week 1
- Build 100-prospect list.
- Send 40 targeted messages.
- Create demo dataset.
- Create landing page.
- Book 8 interviews.
- Build first dashboard mock.

## Week 2
- Complete 10-15 interviews.
- Show sample report to 5 people.
- Run first demo walkthroughs.
- Ask first price question: "$10k-$25k assessment?"

## Week 3
- Complete 10 more interviews.
- Improve demo.
- Test Graph sandbox.
- Ask for design partner.
- Identify trust blockers.

## Week 4
- Complete 30 total interviews.
- Get 3 pricing conversations.
- Get 1-2 serious pilot next steps or kill.
- Decide: build, narrow, or stop.

## Personas
CISO, Head of Security, M365 admin lead, SharePoint admin, security architect, data governance lead, AI governance lead, Power Platform admin.

## Do not ask
"Would this be useful?"

## Ask
"What happened the last time security reviewed Copilot rollout?"
"What did you have to prove?"
"What tool/process did you use?"
"What budget would pay for this?"
"What would make you buy this quarter?"
