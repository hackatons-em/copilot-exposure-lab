# Dashboard Copy

## Header
Copilot exposure assessment

## Subtitle
Safe exposure testing for Microsoft 365 Copilot, SharePoint, OneDrive, and Copilot Studio agents.

## Buttons
- Run exposure assessment.
- Export report.
- Review evidence.
- Mark as remediating.
- Rerun scenario.

## Severity labels
Critical: likely requires immediate remediation.
High: sensitive data or risky action is reachable.
Medium: governance weakness that could become exposure.
Low: limited hygiene issue.
Info: context for review.

## Finding examples
- Sensitive compensation file accessible through organization-wide link.
- Customer contract folder exposed to broad sales group.
- Board document lacks label and owner.
- Guest user retains access to confidential project site.
- Helpdesk agent can send sensitive summaries externally.

## Trust copy
Metadata-only mode is enabled. Document contents are not stored.
Risk scores are deterministic and evidence-backed.
