# Visual Style

## Direction
Enterprise security product. Light, calm, precise.

## Layout
- Clean white/light neutral base.
- Dense but readable tables.
- Cards for key metrics.
- Side panels for evidence.
- Minimal animation.

## Typography
System sans-serif. Monospace for IDs, paths, evidence chains.

## Color
Use severity colors sparingly. No loud cyber gradients.

## Components
- Severity badge.
- Evidence timeline.
- Permission path graph.
- Finding table.
- Remediation card.
- Report preview.
- Permission consent card.

## Hero screenshot
Show critical findings, evidence chain, remediation, and export report button.
