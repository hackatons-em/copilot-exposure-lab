# Pilot SOW Template

## Pilot name
Copilot Exposure Assessment

## Objective
Run a controlled metadata-first assessment to identify Microsoft 365 Copilot and Copilot Studio exposure paths in approved scope and deliver an evidence-backed remediation report.

## Scope

- Selected M365 scope.
- SharePoint/OneDrive sites.
- Metadata and permission analysis.
- Exposure scenarios.
- Findings dashboard.
- Report.
- Findings review call.

Excluded:
- Full content unless explicitly added.
- Automatic remediation.
- Production continuous monitoring.
- Compliance certification.

## Timeline
Week 1: setup and scan.
Week 2: findings and report.
Optional: remediation review and re-scan.
