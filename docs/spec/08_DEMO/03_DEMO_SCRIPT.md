# Demo Script

## 0:00-0:30
"Companies are rolling out Microsoft 365 Copilot, but Copilot can only be as safe as the permissions underneath it. If SharePoint is overshared, Copilot can make exposure easier to discover."

## 0:30-1:00
"This is a fake Microsoft 365 company with HR, legal, finance, security, and project files."

## 1:00-1:45
Run exposure assessment: Normal employee scenario, Bob Product Manager, SharePoint + OneDrive, metadata-only.

## 1:45-3:15
Open critical finding. Show evidence chain: Bob -> Everyone group -> org-wide link -> salary file.

## 3:15-4:15
Show remediation and report export.

## 4:15-5:00
"The buyer does not need another AI dashboard. They need to know whether Copilot rollout creates exposure, which paths matter, and what to fix this week."

## After-demo questions
- Which finding felt real?
- What evidence is missing?
- Would this report help a security review?
- Who would approve a pilot?
